P.E.Op.S. SPU PSX Plugin 1.10 (win32)


In the hiatus of Pete Bernett, the community has
taken the opportunity to repair, upgrade and manage
his previous PEOpS SPU plugin 1.9 (2004)
http://www.pbernert.com


In short, nearly the whole plugin core is rebuilt to
create highly compatible, quality-rich and customizable
features.


Lots of new games work and psx-quality output is
generations better.


As Pete would put it:
"Vagrant Story has never sounded better"



Features:
- Rewrote Async, Timing, ADSR, Noise, IRQ, VAG, XA, CDDA, Decoded buffer engines
- Rewrote Output drivers (DSound, XAudio 2), Save state support
- Partial rewrites of Reverb, Dma, (Gaussian) Interpolation, FMod, Envelope cores
- Added pseudo-5.1, 7.1 support, xa/cdda/output filters, latency/volume control
